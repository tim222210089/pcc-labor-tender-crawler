# 我的特休總覽

原生 Android App，用 Kotlin + Jetpack Compose 製作，資料以 Room 保存在手機本機。

## 已實作功能

- 首頁儀表板
  - 明顯大標題
  - 特休剩餘小時與「幾天又幾小時」換算
  - 加班補休剩餘與 30 天內到期提醒
  - 各假別總額、已請、剩餘、進度條

- 特休
  - 每年可自訂特休總天數
  - 自動以 1 天 = 8 小時換算
  - 年度可切換

- 加班補休
  - 可新增補休時數
  - 可設定到期日
  - 請特休時可選擇扣特休或扣加班補休
  - 可編輯補休批次的加班日期、到期日、總時數與備註
  - 未使用的補休批次可刪除
  - 已被請假使用的補休批次不可刪除，總時數也不可改小於已使用時數

- 請假
  - 支援全天 8 小時、半天 4 小時、自訂整數小時
  - 新增請假可選開始日期與結束日期，系統會為範圍內每個工作日建立紀錄
  - 週六週日不可作為請假日期
  - 範圍請假會自動跳過週六週日
  - 同一天最多 8 小時
  - 額度不足時不可儲存
  - 可在「紀錄」頁編輯或刪除既有請假紀錄
  - 刪除前會出現確認視窗

- 不可休設定
  - 可設定整天不可休
  - 可設定上午不可休
  - 可設定下午不可休

- 請假紀錄
  - 可依假別篩選，查看各假別休在哪一天
  - 首頁點選假別卡片可直接打開該假別日期明細
  - 扣加班補休的紀錄會保存補休批次明細，刪除或編輯時會正確回補
  - 紀錄頁提供「行事曆」與「清單」兩種模式
  - 行事曆會用粉紅色標記週六、週日與國定假日
  - 有請假的日期會直接顯示短假別名稱，例如特休、事假、病假
  - 同一天多筆請假時最多顯示 2 筆，其餘以 +N 表示
  - 不可休日期在月曆格內以短標籤顯示，避免文字換行撐高框框
  - 點選日期後可看當日請假與不可休明細
  - 可手動新增或刪除自訂國定假日
  - 可按「更新國定假日」從政府開放資料線上同步，失敗時保留本機資料

## 假別範圍

- 特休
- 加班補休
- 事假
- 家庭照顧假
- 身心調適假
- 病假
- 生理假
- 婚假
- 產前假
- 娩假
- 流產假
- 陪產檢及陪產假
- 喪假

公假不納入。

## 開發方式

用 Android Studio 開啟此資料夾：

```text
C:\Users\tim\Desktop\codex code\take time off
```

等待 Gradle 同步後執行 `app`。

目前已使用 Android Studio 內建 JBR 驗證：

```text
.\gradlew.bat :app:assembleDebug
BUILD SUCCESSFUL
```
